### Program:16 Write a program to show the work of float and double
```C
#include <stdio.h>
Int main()
{
 float mark1 = 94.678;
 double mark2 = 91.32;
 printf(“Niko scored = %f\n”, mark1);
 printf(“Cassie scored = %lf”, mark2);
  return 0;
}

### OUTPUT:
Niko scored 94.678000
Cassie scored 91.320000
```
